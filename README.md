
# 🎬 Movie-Club: The Ultimate Cinematic Experience 🚀 [Live](https://movie-club-gp.vercel.app/)

### 🎥 Discover the perfect movie or TV show

✨Movie-Club is a new platform for cinema lover. With our powerful search and filtering tools, you can easily find the perfect movie or TV show to watch, no matter what you're in the mood for.


### 👉Here are some of the features that make Movie-Club the ultimate cinematic experience: 👍

- ***🔎Smart Search:*** Find your favorite movies and TV shows effortlessly with our intuitive search functionality.

- ***🎞Advanced filtering:*** Narrow down your search results with over 12 filters, including language, country, and streaming platform and more

- ***🔁Real-time data updates:*** Always get the latest information on movies and TV shows, including ratings, reviews, and availability.

- ***✨Lazy loading:*** Load images and React components only when they're needed, for a faster and more responsive experience. 

- ***🖱Infinite scrolling:*** Scroll through our vast catalog of movies and TV shows without ever having to worry about running out.

- ***🌟Ratings and reviews:*** Rate and review movies and TV shows to share your thoughts with the community.

- ***👫Sharing:*** Easily share your favorite movies and TV shows with friends and family

- ***🌐SEO optimization:*** Get found by more people with our SEO-friendly website.

- ***❤Favorites and watchlists:*** Save your favorite movies and TV shows for easy access.

- ***📱Responsive design:*** Enjoy Movie-Club on any device, from your desktop to your smartphone.

- ***✨Debouncing:*** Prevent unnecessary server requests for a smoother user experience.

- ***✨In-Depth Insights:*** Get a closer look at each movie or TV show. Explore cast and crew details, reviews, and discover similar titles.

- ***✨Trailer Extravaganza:*** Dive into the world of trailers! Watch previews for movies and TV shows directly on Movie Club

- ***💿Local storage:*** Your preferences will be saved even after you reload the page.

- ***🎇Tailwind CSS:*** Stunning visuals powered by the magic of Tailwind CSS.

## How to Use

- Clone the repository: git clone https://github.com/Ghanshyam-parmar96/Movie-club.git

- Install dependencies: npm install

- Create a .env file in the root directory and add your TMDB API key:

Note: You can obtain your TMDB API key by creating an account at https://www.themoviedb.org/documentation/api

- Install dependencies
```bash
  npm install
```
- Start the locally 
```bash
  npm run dev
```
